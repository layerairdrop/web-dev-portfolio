5 Kesalahan SEO yang Sering Dilakukan Website Baru

Website baru sering gagal di Google karena kesalahan dasar SEO. Kami bahas 5 kesalahan paling umum dan cara fix-nya. Plus checklist SEO on-page gratis.

Kesalahan #1: Tidak Ada Meta Description

Meta description adalah cuplikan 150-160 karakter yang muncul di hasil Google. Tanpa ini, Google akan mengambil random text dari halaman kamu — biasanya hasilnya jelek dan tidak menarik klik.

<!-- BENAR -->
<meta name="description" content="Klinik Sehat Sentosa — layanan kesehatan 
keluarga di Jakarta Selatan. Booking online, dokter spesialis, lab klinik. 
Hubungi: 021-XXXX-XXXX.">

<!-- SALAH: kosong atau auto-generated -->
<meta name="description" content="Home page of My Website">

Kesalahan #2: Gambar Tanpa Alt Text

Google tidak bisa "melihat" gambar — dia membaca alt text. Tanpa alt text, gambar kamu invisible di Google Image Search (yang bisa bawa 20-30% traffic).

<!-- BENAR -->
<img src="produk-1.jpg" alt="Kemeja batik solo warna biru ukuran L">

<!-- SALAH -->
<img src="IMG_20260701_001.jpg">

Kesalahan #3: Tidak Mobile-Friendly

Google pakai mobile-first indexing — mereka rank berdasarkan versi mobile, bukan desktop. Website yang tidak responsif otomatis kalah di ranking.

Cara cek: buka website di HP. Kalau harus zoom untuk baca teks, atau tombol terlalu kecil untuk di-tap, itu masalah. Minimum ukuran teks: 16px. Minimum tombol: 44x44px.

Kesalahan #4: Load Time >3 Detik

53% pengunjung bounce kalau website load lebih dari 3 detik. Google juga menggunakan Core Web Vitals sebagai ranking factor.

MetricTargetCara Cek
LCP (Largest Contentful Paint)<2.5 detikPageSpeed Insights
FID (First Input Delay)<100msChrome DevTools
CLS (Cumulative Layout Shift)<0.1Lighthouse

Quick wins untuk speed:

  • Kompres gambar (WebP format, max 200KB per gambar)
  • Lazy load gambar di bawah fold
  • Minify CSS dan JS
  • Gunakan CDN (Cloudflare gratis)
  • Hapus plugin/JS yang tidak dipakai

Kesalahan #5: Tidak Ada Internal Linking

Internal link membantu Google memahami struktur website dan mendistribusikan authority antar halaman. Tanpa internal link, halaman deep di website kamu jarang ter-index.

<!-- BENAR: kontekstual internal link -->
<p>Lihat juga <a href="/layanan/klinik-umum">layanan klinik umum</a> 
kami untuk informasi lebih detail.</p>

<!-- SALAH: "klik di sini" -->
<p>Untuk info lebih lanjut, <a href="/layanan">klik di sini</a>.</p>
💡 Bonus: Checklist SEO On-Page
  • ☐ Title tag 50-60 karakter dengan keyword utama
  • ☐ Meta description 150-160 karakter dengan CTA
  • ☐ H1 tag satu per halaman
  • ☐ Alt text di semua gambar
  • ☐ Internal link minimal 3 per halaman
  • ☐ Mobile responsive
  • ☐ PageSpeed >90
  • ☐ HTTPS aktif
  • ☐ Sitemap.xml submitted ke Google Search Console
  • ☐ Robots.txt tidak blok halaman penting
