## Hero Section Patterns

### Pattern A: Image Slider (Enseval, Kimia Farma)
- Full-width carousel with 3-6 slides
- Each slide: image + overlay text + CTA
- Auto-rotate with manual controls
- **Best for**: Large companies with multiple messages

### Pattern B: Split Layout (Alkespedia)
- Left: Text content (H1 + description + CTA)
- Right: Product/equipment image
- Clean whitespace between sections
- **Best for**: Conversion-focused, single-message hero

### Pattern C: Video Background (GE HealthCare)
- Full-width video loop
- Overlay text with semi-transparent background
- Play/pause controls
- **Best for**: Premium brands, storytelling

### Pattern D: Static Image + Overlay (Philips)
- High-quality hero image
- H1 text overlay
- Minimal design, maximum impact
- **Best for**: Brand-focused, visual storytelling

### Recommended for PT Maju Bersama:
**Pattern B (Split Layout)** - Most appropriate for a local distributor:
- Left side: Company name + tagline + "Konsultasi Gratis" CTA
- Right side: Medical equipment image or team photo
- Mobile: Stack vertically (text on top)

## Mobile Responsiveness Patterns

### Observed Mobile Behaviors:
1. **Navigation**: Hamburger menu → full-screen overlay
2. **Hero**: Stack vertically, reduce text size
3. **Stats**: 2x2 grid instead of 4-column
4. **Product Cards**: 1-2 columns instead of 3-4
5. **Contact Form**: Full-width, stacked inputs
6. **WhatsApp Button**: Fixed bottom-right, always visible
7. **Images**: Lazy loading, responsive sizes

### Mobile-First Recommendations:
- Touch-friendly tap targets (min 48px)
- Click-to-call phone numbers
- Click-to-chat WhatsApp links
- Sticky header with CTA
- Bottom navigation bar (optional)
- Simplified menu structure
