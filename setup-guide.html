Setup Bot LYID

Panduan lengkap dari download hingga bot berjalan. Ikuti step by step.

1
Download & Extract

Setelah membeli bot, kamu akan menerima file .zip atau akses ke GitHub repo private.

Langkah

  1. Download file dari link yang diberikan @OG_LYID_bot
  2. Extract ke folder yang mudah diakses (contoh: C:\LYID-BOTS\peakai-bot\)
  3. Pastikan semua file ada: package.json, src/, config.json.example, README.md
Jika dapat akses GitHub, clone dengan: git clone https://github.com/layerairdrop/[bot-name].git
2
Install Dependencies

Bot butuh Node.js dan beberapa package. Install semua dependency yang diperlukan.

Prasyarat

  • Node.js v18+ — download dari nodejs.org
  • Git (opsional) — untuk clone dari GitHub

Install

cd path/to/bot-folder
npm install
Jika ada error, coba: npm install --force atau hapus node_modules/ lalu install ulang.
3
Configure

Copy file contoh config dan isi dengan data kamu.

Langkah

cp config.json.example config.json

Buka config.json dengan text editor (Notepad++, VS Code, dll). Isi:

  • Wallet address — wallet Ethereum/EVM kamu
  • Private key (opsional) — untuk on-chain TX
  • Proxy (opsional) — format: http://user:pass@ip:port
  • Referral code — jika ada
  • API keys — Capsolver, 2captcha (jika butuh captcha)
JANGAN share config.json ke siapa saja. File ini berisi data sensitif.
4
Run Bot

Jalankan bot dan pastikan berjalan dengan baik.

Run

node index.js

Atau untuk beberapa bot Python:

python bot.py

Run Background (24/7)

npm install -g pm2
pm2 start index.js --name my-bot
pm2 save
Gunakan PM2 untuk menjalankan bot 24/7. PM2 akan auto-restart jika bot crash.

Monitor

pm2 status          # cek status semua bot
pm2 logs my-bot     # lihat log real-time
pm2 restart my-bot  # restart jika error
5
Verify & Monitor

Pastikan bot berjalan dengan benar dan monitor hasilnya.

Checklist

  • ☐ Bot started tanpa error
  • ☐ Task pertama completed
  • ☐ Reward tercatat (cek di dashboard project)
  • ☐ Log menunjukkan progress
  • ☐ Tidak ada rate limit / ban

Common Issues

  • "Cannot find module" → npm install ulang
  • "ECONNREFUSED" → cek proxy / internet
  • "Rate limit" → tambah delay di config
  • "Invalid license" → hubungi admin
