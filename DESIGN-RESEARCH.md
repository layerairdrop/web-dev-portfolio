# Design Research — Best Practices per Industry (2025-2026)

> Research dari Awwwards, Framer, Dribbble, Colorlib, Saaspo, dan design blogs terkemuka.

---

## 1. RESTORAN / F&B

### Referensi Website Terbaik:
- NKORA (framer.com) — minimalist, high-end, clean lines
- Nuff — bold typography, electric colors, playful
- Deleito — warm, inviting, food-forward

### Key Design Patterns:
- 77% diners check website before choosing restaurant
- 69% say website is #1 factor in deciding where to eat
- HIGH-QUALITY food photography = MUST (bukan stock photo)
- Menu harus mudah diakses (1-2 klik dari hero)
- Online ordering system integration
- Localized SEO (Google Maps, jam operasional)
- Mobile-first (70%+ traffic dari mobile)
- WhatsApp/telepon CTA prominent

### Rekomendasi untuk demo kita:
- Tambahkan menu PDF download
- Tambahkan online reservation system
- Food photography harus "drool-worthy"
- Jam operasional di hero section

## 2. TECH / SaaS

### Referensi Website Terbaik:
- Linear (linear.app) — ultra-minimal dark, purple accent
- Vercel (vercel.com) — black/white precision, Geist font
- Mintlify — clean, green-accented, reading-optimized
- Raycast — sleek dark chrome, vibrant gradient accents

### Key Design Patterns:
- Hero: single headline + subheadline + CTA (no clutter)
- Dark theme dominant untuk tech/SaaS
- Social proof bar (logos, stats, testimonials)
- 3-tier pricing (Free/Pro/Enterprise)
- Feature grid dengan ikon + deskripsi singkat
- Dashboard preview/screenshot di hero
- "Trusted by" logo bar di bawah hero
- Gradient accent (purple/blue/cyan) untuk CTA buttons

### Rekomendasi untuk demo kita:
- Tambahkan animated dashboard preview
- Tambahkan "Trusted by" logo bar dengan brand real
- Tambahkan comparison table (vs kompetitor)

## 3. CREATIVE AGENCY

### Referensi Website Terbaik:
- Awwwards Design Agencies (awwwards.com/websites/design-agencies)
- Top 100 Most Creative Portfolios (medium.muz.li)

### Key Design Patterns:
- Bold, experimental typography
- Work gallery sebagai hero (show, don't tell)
- Scroll-triggered animations
- Case study pages (project → process → result)
- Client logos + testimonials
- Marquee/scrolling text untuk brand names
- Dark theme dengan accent color vibrant
- Interactive hover effects pada portfolio items

### Rekomendasi untuk demo kita:
- Tambahkan case study detail pages
- Tambahkan scroll animations (Intersection Observer)
- Tambahkan interactive cursor effects

## 4. E-COMMERCE / TOKO ONLINE

### Referensi Website Terbaik:
- Tokopedia, Shopee (Indonesian market leaders)
- Awwwards E-commerce (awwwards.com/websites/e-commerce)

### Key Design Patterns:
- Product grid dengan filter/sort prominent
- Search bar di header (always visible)
- Cart icon dengan badge count
- Product cards: image + name + price + rating
- Quick view / zoom on hover
- WhatsApp checkout populer di Indonesia
- Payment gateway integration (Midtrans, Xendit)
- Free shipping badge = conversion booster
- Product categories di hero/mega menu

### Rekomendasi untuk demo kita:
- Tambahkan search bar di header
- Tambahkan product rating stars
- Tambahkan quick view modal
- Tambahkan free shipping badge

## 5. CRYPTO / WEB3

### Referensi Website Terbaik:
- Turum-burum Web3 examples
- Uniswap, Aave, Compound (DeFi leaders)
- ChainVault (our demo)

### Key Design Patterns:
- Dark theme dominant (trust, tech feel)
- Dashboard preview dengan live data
- Multi-chain indicators (chain logos/badges)
- Connect Wallet CTA (prominent)
- Stats bar (TVL, users, volume)
- Gradient accents (blue/purple/cyan)
- Security badges (audited, verified)
- Documentation link prominent
- Token/price ticker

### Rekomendasi untuk demo kita:
- Tambahkan animated chart bars
- Tambahkan chain logos (not just colored dots)
- Tambahkan "Audited by" badge

## 6. KLINIK / HEALTHCARE

### Referensi Website Terbaik:
- Framer Healthcare Examples (15 sites)
- WebToMed Best Hospital Designs
- JanBask Healthcare Examples

### Key Design Patterns:
- Clean, trustworthy (blue/white/teal dominant)
- Doctor profiles dengan foto + spesialisasi + jadwal
- Online booking/appointment system
- Service list dengan ikon
- Patient testimonials (dengan foto)
- FAQ section (BPJS, biaya, prosedur)
- Emergency contact prominent (24/7)
- Google Maps embed
- Insurance/payment info
- Mobile appointment booking critical

### Rekomendasi untuk demo kita:
- Tambahkan insurance badge (BPJS, asuransi swasta)
- Tambahkan emergency hotline prominent
- Tambahkan doctor availability calendar
- Tambahkan patient education blog section

## 7. WEDDING / EVENT

### Referensi Website Terbaik:
- Colorlib 34 Best Wedding Planner Websites
- Dribbble Wedding Planner designs
- DreamCo Design Wedding Solutions

### Key Design Patterns:
- Elegant typography (serif headings: Playfair Display, Cormorant Garamond)
- Full-screen hero photo (couple, venue, decoration)
- Rose/gold/ivory/dusty blue palette
- Photo gallery (masonry grid)
- Package tiers (Intimate/Classic/Grand)
- Testimonials dengan foto pengantin
- WhatsApp inquiry prominent
- Countdown timer (to wedding date)
- RSVP form integration
- Venue information + map

### Rekomendasi untuk demo kita:
- Tambahkan countdown timer
- Tambahkan masonry gallery layout
- Tambahkan venue map embed
- Tambahkan RSVP form
